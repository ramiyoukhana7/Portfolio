Pendlaren - Trafiklab API

Motivated by the knowledge I gained from Scrimba, I built a real time trip planning web app. The app seamlessly integrates with the user's current position to provide real-time transit data. Here‚Äôs what it delivers:

Pinpoint Accuracy: Utilizes the geolocation API to obtain the user's precise longitude and latitude.

Intuitive Interface: After fetching the nearby stops through an API call to ReseRobot - travel planner, users can easily select their desired stop.

Real-Time Departure Data: Upon selecting a stop, another API call to ReseRobot - bus time tables is made using the stop's id (from the property extId in the JSON response) to present the user with real time departures.

Journey Planning: Beyond just departure times, users can also conveniently plan trips between two stations, streamlining their entire travel experience.
